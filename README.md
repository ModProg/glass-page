# glass-page

```
USAGE:
    glass-page [FLAGS] [OPTIONS] <url>

ARGS:
    <url>

FLAGS:
    -c, --clear      Trys to make the Backround Transparent via CSS
        --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -h, --height <height>    [default: 500]
    -t, --title <title>      Set custom Title [default: Glass Page]
    -w, --width <width>      [default: 500]
    -x <x>
    -y <y>
```
